# Open Graph Image Requirements

## Required Images

### 1. Main OG Image (`og-image.jpg`)
- **Size**: 1200 x 630 pixels
- **Format**: JPG or PNG
- **Usage**: Default social sharing image for all pages
- **Content**: 
  - WaveKit logo
  - Tagline: "AI-Powered Audio Editing Tools"
  - Visual elements: Audio waveforms, music notes
  - Brand colors: Ocean Teal (#14b8a6) and Coral Orange (#f97316)

### 2. Logo (`logo.png`)
- **Size**: 512 x 512 pixels (square)
- **Format**: PNG with transparent background
- **Usage**: Organization logo for structured data
- **Content**: WaveKit logo icon

### 3. Favicon (`favicon.ico`)
- **Size**: 32 x 32 pixels (and 16 x 16)
- **Format**: ICO
- **Usage**: Browser tab icon

## Design Guidelines

### Color Palette
- **Primary**: Ocean Teal (#14b8a6)
- **Accent**: Coral Orange (#f97316)
- **Background**: White or light gradient
- **Text**: Dark gray (#1f2937)

### Typography
- **Font**: Inter or similar modern sans-serif
- **Hierarchy**: Clear title and subtitle

### Visual Elements
- Audio waveforms
- Musical notes
- Microphone icons
- Sound wave patterns
- Gradient backgrounds

## Tools for Creating Images

1. **Canva** (https://canva.com)
   - Use template: "Facebook Post" (1200 x 630)
   - Add brand colors and elements

2. **Figma** (https://figma.com)
   - Professional design tool
   - Export at 2x for high quality

3. **Adobe Express** (https://express.adobe.com)
   - Quick social media graphics
